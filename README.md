# Data Cleaning Tool (Project 1)

## Overview
This project focuses on understanding and cleaning a real-world dataset using python and pandas. The goal is tho build a clean, reliable dataset that can be used for analysis or machine learning tasks.

Each step is done incrementally, following a professional data workflow.

---

## Dataset
- Source: Netflix Movies and TV Shows dataset
- Each row represents a movie or TV show
- the dataset contains metadata such as title, director, cast, country, release year, ranting, and description

___

## Work Done So Far
- Loaded the dataset using Pandas
- Verified dataset shape and columns
- Inspected sample rows to understand data structure
- Identified data types for each columns
- Analyzed missing values across columns

No data cleaning has been applied yet.
The current focus is on understanding the dataset before making changes.

## Tech stack
- Python
- Pandas
